配置Hexo

发表于 2016-04-26   |   分类于 学习笔记 ， hexo学习   |  

hexo要在node环境下部署。
主要命令：

  1. 装置hexo
    $ npm install -g hexo
  2. 部署hexo
    $ hexo init
  3. 生成html页面
    $ hexo g
  4. 运行测试环境
    $ hexo s
    接下来就可以在浏览器中输入 localhost:4000 查看了
  5. 安装部署插件
    $ npm install hexo-deployer-git--save

  6. 配置_config.yml文件，在文件的最后一行加上

    deploy:
  type: git
  repo: https://github.com/XXX/XXX.github.io.git(你的git地址)
  branch: master

  7. 部署

    $ hexo d

另外type的值也有可能为github。

当然，在进行第五步部署到git上之前，必须得有一个git地址。
